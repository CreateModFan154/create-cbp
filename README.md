## Overview

**Create: CBP** is an **in-development** Minecraft mod that introduces **Customs and Border Protection (CBP)** systems to your world. This mod allows players to set up automated **border stations**, process **cargo**, and **detect contraband** using the mechanics from the **Create mod**. While the mod is **still in development**, the goal is to create an immersive **border control** experience with **cargo scanning**, **contraband detection**, and **border patrol NPCs**.
Crowdin: https://crowdin.com/project/create-cbp

## Features (WIP)

### Core Features (Planned):

* **Contraband Detection**: Identify and process illegal goods, including items like **"Blue Stuff"** (meth), **"Bag of 'Sugar'"** (coke), and **"Fake Money"**.
* **Automated Border Stations**: Use **Create mod** blocks (e.g., **conveyor belts**, **mechanical arms**, and **pistons**) to build and automate inspection stations.
* **NPC Guards**: Patrol the border and check for contraband. NPCs will potentially issue **fines**, **confiscations**, and even **arrests**.
* **Penalties**: Smuggling illegal goods may result in **fines**, **confiscations**, or **penal actions**. Players caught smuggling contraband will face consequences.

### Features in Development:

* **Contraband Types**: Current focus on **Blue Stuff**, **Bag of 'Sugar'**, and **Fake Money**. More contraband types like **Crack**, **Illegal Weapons**, and **Stolen Artifacts** are planned.
* **NPC Interactions**: NPCs will eventually interact with players and enforce border control rules, though this feature is **still under development**.
* **Vehicle Inspection**: Integration with **IV mod** (Immersive Vehicles) planned for vehicle cargo processing. This will allow for more immersive cargo checks.
* **Expanded Processing Systems**: Working on creating more intricate cargo scanning and detection mechanics.
